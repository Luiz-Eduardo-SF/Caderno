Sobre Positions

Position Static

A position: static; é a posição padrão de um elemento, ou seja, por se tratar de uma posição padrão, ela nem precisa ser declarada.

Position Static

Position Relative

A position: relative; define uma posição a partir da sua posição natural (static).

Position Relative

Position Absolute

A position: absolute; posiciona o elemento em relação ao elemento pai posicionado ou, na ausência deste, em relação à tag <body>.

Position Absolute

Position Fixed

A position: fixed; posiciona o elemento em relação ao viewport(tela).

O cabeçalho desse site é o próprio exemplo de posição fixed

Position Sticky

A position: sticky; altera entre position: relative; e position: fixed;, permanecendo no lugar definido até que o scroll ultrapasse esse ponto, quando então fica fixa na tela.

Position Sticky

Viu que legal?? É importante que o elemento pai do elemento de sticky tenha uma altura ou largura sulficiente para que o elemento possa se mover também.
