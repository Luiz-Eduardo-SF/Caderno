Sobre as Tags

Nessa aula eu estou utilizando diversas tags

  • <main> - Usada para definir o conteudo principal da pagina
  • <div> - Usada para definir um bloco de conteudo
  • <code> - Usada para definir um trecho de codigo
  • <hr> - Usada para defini uma linha horizontal

Estilizando com CSS

Explicando os estilos e as tags do site!!!

  • div.box - Aplica o estilo nas tags <div> que faz parte da classe "box"
  • *{} - Aplica o estilo em todas as tags
  • margin - Aplica um expaçamento externo na tag
  • padding - Aplica um expaçamento interno na tag
  • background-color - Aplica uma cor de fundo
  • box-shadow - Aplica uma sombra na tag
  • width - Altera a largura da tag
  • height - Altera a altura da tag
  • display - Altera o comportamento da tag
  • flex-direction - Aplica uma direção na tag
  • justify-content - Aplica um alinhamento na horizontal
  • align-items - Aplica um alinhamento na vertical
  • border-radius - Aplica uma borda arredondada
  • color - Muda a cor dos elementos (textos, linhas, etc)
