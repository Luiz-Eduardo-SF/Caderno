Cores e fontes

Cores

Como vimos na anotação dessa aula, nós podemos chamar as cores de diversas formas em CSS:

Chamada Resultado
rgb(255, 0, 0)
rgba(255, 0, 0, 0.5)
hsl(120, 100%, 50%)
hsla(120, 100%, 50%, 0.5)
#ff09ff
blue

Gradientes

Aqui estão alguns tipos de gradientes que podemos fazer em CSS:

Chamada Resultado
linear-gradient(#ffcc00, #ff4f0f)
linear-gradient(90deg, #ffcc00, #ff4f0f)
radial-gradient(#ffcc00, #ff4f0f)

Isso é só alguns comandos básicos. Para saber mais sobre gradientes, clique aqui.

Fontes

Alguns comandos CSS para configurar fontes:

Chamada Resultado
font-family: 'Arial', 'Poppins', sans-serif; Esse comando define uma família de fontes. Caso a primeira fonte não funcione, ela passa para a segunda e assim por diante.
font-size: 20px; Define o tamanho da fonte.
font-weight: bold; Define o peso da fonte.
font-style: italic; Define o estilo da fonte.

Se quiser saber um pouco mais sobre fontes, clique aqui.
